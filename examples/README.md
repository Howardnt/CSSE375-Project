## Examples

This folder contains **non-production reference code** used to learn APIs and prototype ideas.

### `examples/asm`

- **Purpose**: Keep the provided ASM sample linter code for learning and quick experiments.
- **Note**: Code under `examples/` is **not compiled** by Maven for the main project.

To run the ASM sample manually, you can copy it into a scratch Maven project, or temporarily add it
under `src/main/java` locally (but do not commit that change).
